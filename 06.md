# plotting

make a plot

``` r
twopoint %>%
  count(posteam, two_point_conv_result) %>% 
  mutate(posteam = fct_reorder(posteam, n, sum)) %>% 
  ggplot() +
  geom_col(aes(x=n, y = posteam, fill = two_point_conv_result))
```
